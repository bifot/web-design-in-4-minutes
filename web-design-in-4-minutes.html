Веб-дизайн за 4 минуты

Jeremy Thomas / Михаил Семин

Предположим, что у вас есть продукт, портфолио или просто идея, которой вы хотите поделиться со всеми на своем сайте. Перед тем, как опубликовать продукт в интернет, его нужно сделать максимально привлекательным и красивым, или, хотя бы, прилично выглядившим.

Первый шаг, который мы обязательно должны сделать в процессе создания сайта.

Контент

Основная цель дизайна — расширить представления о контенте. Это может показаться очевидным, но контент — наиболее важный элемент на сайте.

Давайте представим, что вы уже завершили работу по написанию контента, и только что создали пустой CSS-файл style.css. Что в нем нужно прописать первым делом?

Центрирование

Очевидно, что длинные строчки на всю страницу читать тяжело. Если мы расположим текст по центру и ограничим его ширину, то мы повысим читаемость и привлекательность контента. 

body {
  margin: 0 auto;
  max-width: 50em;
}

После центрирования блока с текстом, нам необходимо оформить сам текст.

Шрифт

По умолчанию в браузер стоит шрифт «Times», который выглядит крайне непривлекательно и ужасно (разумеется, не всегда). Изменения шрифта сразу же приукрасит вашу страницу. 

body {
  font-family: "Helvetica", "Arial", sans-serif;
}

Если вы хотите использовать шрифт с засечками, можете попробовать шрифт «Georgia».

Когда мы сделали текст более привлекательным, необходимо сделать еще его и хорошо читаемым.

Отступы

Когда страница кажется пользователю «сломанной», то чаще всего проблема именно в отступах. Правильные отступы обеспечивают пространство как вокруг, так и внутри вашего контента, делая его более читабельным и привлекательным. 

body {
  line-height: 1.5;
  padding: 4em 1em;
}

h2 {
  margin-top: 1em;
  padding-top: 1em;
}

Мы сильно изменили наш веб-сайт, давайте теперь разберем более тонкие детали.

Цвета & контраст

Черный текст на белом фоне может не очень понравиться вашим глазам. Выбирая более мягкий оттенок черного, страница становится более комфортной для ваших глаз и чтения в целом. 

body {
  color: #555;
}

Сохраняя правильный уровень контраста, необходимо выбрать оттенок темнее для важных слов (заголовков, текста жирного начертания).

h1,
h2,
strong {
  color: #333;
}

Наконец-то большая часть страница стала хорошо выглядеть, однако некоторые элементы (к примеру, фрагмент кода) выглядят неуместно.

Баланс

Написав пару строчек в style.css, мы добавляем баланса нашей странице:

code,
pre {
  background: #eee;
}

code {
  padding: 2px 4px;
  vertical-align: text-bottom;
}

pre {
  padding: 1em;
}

На этом этапе, думаю, вы хотите выделить вашу страницу и придать ей индивидуальности.

Основной цвет

У большинства сайтов имеется основной цвет, который необходим для визуального акцента. Этот акцент может быть использован, например, для выделения ссылок.

a {
  color: #e81c4f;
}

Но для сохранения баланса необходимо будет добавить еще несколько дополнительных цветов.

Вторичные цвета

Цветовой акцент может быть дополнен более нежными оттенками, которые будут использоваться для обводки, фона или даже основного текста.

body {
  color: #566b78;
}

code,
pre {
  background: #f5f7f9;
  border-bottom: 1px solid #d8dee9;
  color: #a7adba;
}

pre {
  border-left: 2px solid #69c;
}

Изменив оттенки, почему бы не изменить облик всей страницы...

Пользовательский шрифт

Как говорилось ранее, контент является важнейшей частью веб-страницы, поэтому, используя пользовательский шрифт, мы добавим уникальность вашему сайту.

Используя онлайн-сервисы (например, Typekit), можно «играться со шрифтами». Предлагаю использовать шрифт «Roboto» из сервиса Google Fonts:

@import 'https://fonts.googleapis.com/css?family=Roboto:300,400,500';

body {
  font-family: "Roboto", "Helvetica", "Arial", sans-serif;
}

После того, как мы уникализировали свою веб-страницу, как насчет того, чтобы добавить еще тысячу элементов...

Различная графика и иконки могут использоваться в качестве украшений для поддержки вашего контента, или привлекать внимание в сообщении, которое вы хотите передать пользователю.

Давайте украсим наш header с помощью красивого изображения из сервиса Unsplash. 

header {
  background-color: #263d36;
  background-image: url("header.jpg");
  background-position: center top;
  background-repeat: no-repeat;
  background-size: cover;
  line-height: 1.2;
  padding: 10vw 2em;
  text-align: center;
}

Еще можно добавить логотип.

header img {
  display: inline-block;
  height: 120px;
  vertical-align: top;
  width: 120px;
}

Не будем упускать возможность прибавить уникальности и стилей для нашего текста.

header h1 {
  color: white;
  font-size: 2.5em;
  font-weight: 300;
}

header a {
  border: 1px solid #e81c4f;
  border-radius: 290486px;
  color: white;
  font-size: 0.6em;
  letter-spacing: 0.2em;
  padding: 1em 2em;
  text-transform: uppercase;
  text-decoration: none;
  transition: none 200ms ease-out;
  transition-property: color, background;
}

header a:hover {
  background:  #e81c4f;
  color: white;
}

Вуаля!

Мы разработали достойную страницу за несколько минут, соблюдая основные принципы веб-дизайна. Остается сделать только одну вещь...
